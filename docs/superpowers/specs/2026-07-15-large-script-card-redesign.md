# 团队空间大尺寸脚本卡

## 目标

将团队空间的脚本卡从紧凑三列信息块改为作品集式大卡，解决窄卡片中的文本截断、操作拥挤和视觉层级不清的问题。

## 布局

- 团队空间脚本卡以 `MpPluginCard.vue` 的集市卡片为唯一布局基准：顶部标题带、两行摘要与标签、辅助信息、底部操作四个连续区域。
- 移除右侧身份面板、右侧预留空间和所有相关装饰；卡片内容全宽排布。
- 网格与集市卡片一致：宽屏三列、中等宽度两列、窄屏单列。卡片最小高度约 254px。
- 顶部展示图标、标题和分类；摘要最多两行，语言和标签在摘要下方。
- 辅助信息保留大小、上传者和更新时间；底部保留详情与团队空间原有的管理操作，再保留下载和安装到客户端两个主操作。

## 保留行为

- 不改变下载验证码、配额、安装状态、错误信息、路由、权限判断或组件事件。
- 所有管理操作继续按现有 props 控制可见性。
- 移动端使用单列，操作按钮保持可点击并防止文字截断。

## 验证

- 检查 360px 及更宽卡片下的标题、作者和更新时间没有相互挤压。
- 在团队空间和个人空间确认网格、下载、安装、详情与管理操作均可用。
- 运行组件级 lint 和开发构建；记录与本次改动无关的既有 lint 问题。
