# 浅色主题纯色按钮设计

## 目标

优化浅色主题主操作按钮的视觉效果：移除橙色渐变、光晕和位移动效，改用饱和度适中、层级更稳的平衡橙纯色，同时保持按钮状态清晰、文字易读。深色主题不在本次调整范围内。

## 视觉方案

- 默认背景使用 `#e77732`，悬停背景使用 `#d76724`，按下背景使用 `#bf571c`。
- 按钮文字继续使用白色，保持与杏橙背景的识别度。
- 浅色主题下的主操作按钮不使用渐变和发光阴影，也不在悬停或按下时产生位移。
- 状态切换仅过渡 `background-color`、`border-color` 和必要的 `opacity`，时长统一为约 `160ms`。
- 禁用状态沿用现有透明度和不可点击规则，不新增交互行为。

## 实现范围

- 在浅色主题的共享变量中提供纯色主按钮背景及悬停、按下状态颜色。
- 将使用共享橙色渐变的主操作按钮统一接入纯色状态，避免逐组件产生颜色差异。
- 将“安装到客户端”按钮从独立的 `--accent`、亮度滤镜和下压动效迁移到同一套浅色主题主按钮状态；浅色主题加载时只保留旋转图标，不显示扫光。
- 保留深色主题现有渐变样式；页面背景、卡片、图标和非按钮装饰不受影响。
- 不修改组件业务逻辑、事件、文案、权限判断或响应式布局。

## 验证

- 检查浅色主题下常用主操作按钮的默认、悬停、按下和禁用状态。
- 检查“安装到客户端”与其他主按钮颜色一致，并确认加载状态没有扫光。
- 确认按钮不再出现渐变、发光或位移动效，且状态变化连续自然。
- 切换到深色主题，确认现有按钮视觉保持不变。
- 运行项目现有的 lint 与构建命令，确认样式改动未引入错误。
